You are an expert in success pattern analysis for AI agent systems.

You will be given a collection of extracted **{item_type_label}** from spreadsheet tasks where an AI agent succeeded. Each item is associated with a source case (task IDs).

{item_type_context}

Your task is to compress this collection into a small set of **Pattern Summaries** that maximize coverage (every input item should fall under some pattern) while minimizing the number of patterns. Specifically:
- Group semantically similar items, even if worded differently
- Identify the most frequent and salient patterns
- Preserve traceability to original tasks

The goal is to produce a compact, actionable summary of what success strategies occur most often and which tasks they come from.

### Requirements
1. **Broad Coverage**:
   Every input item must be covered by at least one pattern. Prefer fewer, broader patterns over many narrow ones. Group aggressively — items that share even a partial effective behavior belong together.
2. **Frequency Awareness**:
   Patterns covering more items should be listed first. Rare one-off items should be absorbed into the closest broader pattern rather than forming their own.
3. **Non-Redundancy**:
   Do NOT output overlapping or semantically duplicate patterns. If two candidate patterns share >50% of items, merge them.
4. **Generalization**:
   Each pattern must describe a general mechanism, not a single task-specific detail.
5. **Traceability**:
   For each pattern, report which cases (IDs) contributed to it.
6. **Compression**:
   Produce a small number of patterns that best summarize the whole collection.

---

## Output Format (STRICT)

For each pattern, output:
```markdown
# Pattern i
## Title
<Short name of the pattern>

## Description
<One-sentence explanation of the pattern>

## Frequency
<Count of items covered by this pattern out of total items>

## Source Cases
<Comma-separated task indices only (3–10 examples), e.g., "0, 3, 7">

## Covered Specific Successes
<Bulleted list of the specific effective behaviors the agent demonstrated that fall into this pattern. Each bullet should be a concrete action — not a restatement of the pattern title. If a similar behavior happened twice, feel free to list twice.>
```

You MUST strictly follow the above format when writing Pattern items.
